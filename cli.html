Command-line Interface¶

symbiflow¶

usage: symbiflow [-h] [-v] {all,syn,imp,bit,pgm} ...

SymbiFlow CLI

positional arguments:
  {all,syn,imp,bit,pgm}
                        Available commands
    all                 Performs from synthesis to bitstream generation
    syn                 Performs synthesis
    imp                 Performs implementation
    bit                 Performs bitstream generation
    pgm                 Performs programming

optional arguments:
  -h, --help            show this help message and exit
  -v, --version         show program's version number and exit

symbiflow all¶

usage: symbiflow all [-h] [-t NAME] [--param [PARAM:VALUE [PARAM:VALUE ...]]]
                     [--arch ARCHITECTURE]
                     [--define [DEFINE:VALUE [DEFINE:VALUE ...]]]
                     [--include [PATH [PATH ...]]] [--vhdl FILE[,LIBRARY]
                     [FILE[,LIBRARY] ...]] [--vlog FILE [FILE ...]]
                     [--slog FILE [FILE ...]] [--scf FILE [FILE ...]]
                     [--icf FILE [FILE ...]] [--project PROJECT] [-p FPGA]
                     [-o PATH] [--oci-engine {None,docker,podman}]
                     [--oci-volumes [HOST-DIR:CONT-DIR [HOST-DIR:CONT-DIR ...]]]
                     [--oci-work WORK]

Performs from synthesis to bitstream generation

optional arguments:
  -h, --help            show this help message and exit
  -t NAME, --top NAME   specify a top-level name
  --param [PARAM:VALUE [PARAM:VALUE ...]]
                        specify top-level Generics/Parameters
  --arch ARCHITECTURE   specify a VHDL top-level Architecture
  --define [DEFINE:VALUE [DEFINE:VALUE ...]]
                        specify [System] Verilog Defines
  --include [PATH [PATH ...]]
                        specify [System] Verilog Include Paths
  --vhdl FILE[,LIBRARY] [FILE[,LIBRARY] ...]
                        VHDL files
  --vlog FILE [FILE ...]
                        Verilog files
  --slog FILE [FILE ...]
                        System Verilog files
  --scf FILE [FILE ...]
                        Synthesis Constraint Files
  --icf FILE [FILE ...]
                        Implementation Constraint Files
  --project PROJECT     basename for generated files [symbiflow]
  -p FPGA, --part FPGA  name of the target FPGA part [hx8k-ct256]
  -o PATH, --outdir PATH
                        location for generated files [.]
  --oci-engine {None,docker,podman}
                        OCI engine internally employed [None]
  --oci-volumes [HOST-DIR:CONT-DIR [HOST-DIR:CONT-DIR ...]]
                        volumes for the OCI engine [['$HOME:$HOME']]
  --oci-work WORK       working directory for the OCI engine [$PWD]
